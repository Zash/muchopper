{{ appname() }} does not process any data which hasn’t been purposely made public by the persons responsible to the objects the data refers to.

I don’t want my room to be listed here!

If you want your room to not appear in any public listing, use your client to disable that option for the room. Many clients offer that option by typing /configure in the message input. Refer to your clients manual or support channel, searching for "Room configuration". {{ appname() }} only lists rooms which explicitly allow to be listed. If you room is listed here, chances are it will also be listed elsewhere.

Note that the listing is refreshed only once per hour, so it may take a while for the change to propagete to the web listing.

If you cant your room to appear in public listings in general, but not in this listing, please contact the representative at the end of this page.

Data Processing

{{ appname() }} collects and stores information about chat rooms and domains in the Jabber/XMPP network. In this document, we will refer to these rooms as MUCs (short for "Multi-User Chats"). Only data of persistent MUCs is stored.

Data associated with Chat Rooms

The following data is collected and stored with about a MUC:

  • Its identifying address (Jabber ID, JID).
  • The number of users last seen online in the MUC.
  • The timestamp of the last message seen in the MUC (if available).
  • Whether the MUC can be joined into by everybody (muc_open flag and not muc_passwordprotected flag).
  • Whether the MUC shall be publicly listed (muc_public flag)

In addition, for publicly listed MUCs, the following data is stored:

  • The room description (if available).
  • The room name (if available).
  • The current room subject (if available).

Data associated with domains

Only the domain name is stored for domains. All domains which host MUC services or are referenced (via XEP-0030) are stored.

Data Sharing

{{ appname() }} allows anyone to query all information it has stored about publicly listed MUCs. This is the information seen on this website.

This is merely a convenience service, since Jabber services already expose publicly listed MUCs in directories central to their domain.
